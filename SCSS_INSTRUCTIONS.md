# Инструкции по использованию SCSS

## Установка зависимостей

Для использования SCSS необходимо установить Node.js и npm, затем выполнить:

```bash
npm install
```

## Компиляция SCSS

### Однократная компиляция:
```bash
npm run build-css
```

### Автоматическая компиляция при изменении файлов:
```bash
npm run watch-css
```

## Использование

1. Редактируйте файл `styles/main.scss`
2. Запустите `npm run watch-css` для автоматической компиляции
3. Подключите скомпилированный CSS в `index.html`:

```html
<link rel="stylesheet" href="styles/main.css">
```

## Структура SCSS

- **Переменные** - определены в начале файла
- **Миксины** - переиспользуемые стили (@mixin)
- **Базовые стили** - основные стили для элементов
- **Медиа-запросы** - адаптивные стили

## Примечание

Текущий проект использует встроенные стили в `index.html`. Для перехода на SCSS:
1. Перенесите все стили из `<style>` в `styles/main.scss`
2. Скомпилируйте SCSS в CSS
3. Подключите `styles/main.css` в `index.html`
4. Удалите встроенные стили из HTML
